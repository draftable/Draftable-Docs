---
title: "Launching comparisons directly from file explorer"
description: "Draftable Legal simplifies document comparison by starting comparisons directly from your file explorer's menu. <br /> <br /> File explorer is a file manager application that is included with Microsoft Windows, and examples of how to launch comparisons for both Windows 11 and 10 are shown in this article."
---

## How to launch comparisons directly from file explorer (Windows 11)

<Steps>
    <Step>
    Navigate to the desired folder which contains your desired files to compare  
        <Frame>
        ![mceclip0.png](/images/using-draftable-launching-comparisons-directly/image-1.png)
        </Frame>
    </Step>
    <Step>
    Select the files and **right-click** them, and then click **Show more options**  
        <Frame>
        ![mceclip1.png](/images/using-draftable-launching-comparisons-directly/image-2.png)
        </Frame>
    </Step>
    <Step>
    Click **Compare with Draftable**
        <Frame>
        ![](/images/using-draftable-launching-comparisons-directly/image-3.png)  
        </Frame>
    </Step>
    <Step>
    The New Comparison window will appear with your selected documents  
        <Frame>
        ![](/images/using-draftable-launching-comparisons-directly/image-4.png)
        </Frame>
    </Step>
    <Step>
    Just select the **comparison type** and click **compare** when you're ready!
    </Step>
</Steps>

## Drag and drop

You can also **drag and drop** your files directly from File Explorer into the **New Comparison** window to add documents for your comparison.   
<Frame>
![](/images/using-draftable-launching-comparisons-directly/image-8.png)
</Frame>

Simply click and drag the documents from your file location into each area for the older and newer documents.
